# Tour Website Template

Responsive landing page template for a tour business, built with Next.js and Tailwind CSS. Designed as a single-page site with smooth scroll navigation, image gallery with lightbox, and a contact form.

**[Live Demo](https://tour-website-seven.vercel.app)**

## Sections

- **Hero** — full-viewport background with CTA
- **About** — guide biography with side-by-side layout
- **Tours** — tour cards with duration, price, and booking button (responsive grid)
- **Gallery** — photo grid with hover zoom and lightbox viewer
- **Contact** — contact details, business hours, WhatsApp link, and contact form

## Tech Stack

- **Next.js 16** with App Router, static export
- **React 19**, TypeScript
- **Tailwind CSS 4**
- Deployed on **Vercel**

## Usage

```bash
npm install
npm run dev       # Development server
npm run build     # Static export to /out
```

Customize content in `components/sections/`, swap images and text to fit your business.
